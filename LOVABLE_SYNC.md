# Lovable sync

Этот файл нужен, чтобы форсировать двухсторонний sync GitHub → Lovable.

**Текущее состояние платформы (должно совпадать с Vercel):**
- Весь прогресс обнулён (этап 1 active, остальное locked)
- Демо-данные рейтинга/community убраны
- Уроки, доки и структура курса сохранены
- Коммит-референс: sync after progress wipe

После появления этого коммита в Lovable editor нажми **Publish → Update**, чтобы published URL совпал с Vercel.
